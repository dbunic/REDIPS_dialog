Example shows how to display custom HTML in dialog box. If show() method has "html" as first parameter then it implies that HTML is already set with REDIPS.dialog.html() public method. So, it's needed to define HTML before calling show() method.

Here is how to define HTML:

REDIPS.dialog.html('<strong>This is my HTML</strong>');

And here is how to show it:

REDIPS.dialog.show(200, 200, 'html');

Buttons definition are optional, if needed they can be defined as well:

REDIPS.dialog.show(200, 200, 'html', 'Close');
